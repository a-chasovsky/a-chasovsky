#### Проекты по аналитике данных

**pers_motogp_analytics** <br>
Анализ результатов гран-при мирового чемпионата по мотогонкам MotoGP. По запросу пользователя скрипт автоматически скачивает протокол гонки в формате pdf, используя разделы сайта (от пользователя не требуется знать ничего, кроме названия гран-при и года проведения). Данные извлекаются из протокола и преобразовываются в датафреймы pandas; после этого считаются метрики и строятся итоговые графики. Особенность в том, что в извлеченных данных много "мусора" (сдвоенные или оборванные строки, имена, фразы, ненужная информация). Алгоритм не просто выбирает нужные строки, а способен различать, являются ли они необходимой для анализа инфоррмацией или "мусором".
